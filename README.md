## 🚀 Projects

### 📋 [JobTrack – Job Application Tracker](https://joybarotjb1.github.io/job-tracker)
> A clean, dark-themed web app to track job applications — built with pure HTML, CSS & JavaScript

- ✅ Add & manage job applications with company, role & date
- ✅ Track status: Applied → Interview → Offer / Rejected / Ghosted
- ✅ Live pipeline dashboard with real-time counts
- ✅ Filter by status & add notes
- ✅ Data saved in browser (localStorage)
- ✅ Fully responsive on mobile

**Tech:** `HTML5` `CSS3` `JavaScript` `localStorage`
